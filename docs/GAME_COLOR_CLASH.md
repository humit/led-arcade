# Color Clash MVP

Color Clash is a 2–4 player territory-painting game for the 8×32 matrix.

- Players move continuously in one of four absolute directions.
- Movement wraps at all screen edges.
- Every visited cell is repainted in the player color.
- Simultaneous arrivals on the same cell neutralize it.
- A round lasts 30 seconds.
- The player owning the most cells wins.
- With one human, an automatic CPU opponent is added.
- The controller uses four large directional touch zones and shows no gameplay HUD.
